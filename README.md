# ポイント収益ノート

TikTok Liteなどのポイント活動で使う複数端末の仕入れ額と、毎月のポイント収益を記録するスマホ向けWebアプリです。

## できること

- 端末ごとの仕入れ値、購入日、仕入れ元、状態を記録
- TikTok Liteや他アプリの月次ポイントを端末別に記録
- 累計収益、仕入れ総額、累計利益、今月の収益を自動計算
- 端末ごとの回収率、残り回収額、利益を表示
- JSONバックアップ、JSON復元、CSV出力
- ブラウザのlocalStorageに自動保存

## 使い方

`index.html` をブラウザで開くだけで使えます。GitHub Pagesで公開すると、スマホからURLにアクセスして利用できます。

## GitHub Pages公開

このリポジトリをGitHubへpushすると、`.github/workflows/pages.yml` によりGitHub Pagesへ自動デプロイされます。

公開URLは通常、次の形式です。

```text
https://<GitHubユーザー名>.github.io/<リポジトリ名>/
```

## 注意

データはアクセスした端末のブラウザ内に保存されます。別スマホへ移す場合は、JSONバックアップとJSON復元を使ってください。
